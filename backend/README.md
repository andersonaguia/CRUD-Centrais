## Pré-requisitos

[NodeJS](https://nodejs.org/en) 22+ 

[NestJS](https://nestjs.com/) 11+

## Configuração do projeto

1 - Instale as dependências

```bash
$ npm install
```

## Compilando e rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Rodando testes

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
